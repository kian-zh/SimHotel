# SimHotel 宏大战略路线图

## 愿景

SimHotel 应从酒店大亨 MVP 演进为关于旅行、资本、城市与酒店网络的宏大历史策略游戏。长期目标是接近《欧陆风云》式战役：玩家解读宏观历史、选择战略姿态、建立区域势力、在冲击中生存，并在数十年间塑造全球酒店秩序。

## 当前支柱

- **真实地图扩张**：城市通过一次性开城费解锁，无年份门槛；可通过人口、经济、旅游图层检视。
- **酒店空间经营**：客房与公共设施统一消耗「空间」；扩建增容、技术树解锁高级设施（设计见 [HOTEL_OPERATIONS_DESIGN.md](./HOTEL_OPERATIONS_DESIGN.md)）。
- **酒店详情与并购**：点击酒店打开详情 Dialog；支持扩建、装修、雇人、广告、出售与合理价格收购。
- **每日模拟**：需求、入住率、收入、支出、声誉、进行中事件与 AI 对手通过 Tick 引擎更新。
- **历史事件层**：重大事件暂停游戏；普通事件进入新闻栏并改变市场修正。
- **本地选址**：酒店坐标通过网格采样影响区位质量。
- **集团战略**：玩家可选择全集团政策，改变建造成本、运营成本、市场吸引力与声誉压力。
- **资本市场**：可借贷与还款，杠杆影响信用评级与利息支出。
- **任务式引导**：教程步骤追踪具体操作——查看城市、打开建造、落子、打开战略、阅读新闻。
- **城市档案**：每个市场可展示战略原型、机会、风险与建议姿态。
- **事件影响可读性**：重大事件弹窗与新闻卡片可展示驱动模拟的具体修正效果。
- **扩展世界地图**：欧洲、中东、南亚、东南亚、大洋洲为长线战役提供更多扩张目标。
- **区域态势视图**：宏观仪表盘可按区域展示已解锁城市与玩家资产。

## 下一阶段玩法系统

1. **外交与监管**
   - 城市政府暴露许可严格度、税收减免、 zoning 摩擦与公私合作机会。
   - 玩家可游说、谈判目的地推广活动，或接受本地约束。

2. **品牌组合**
   - 将单一品牌拆分为经济、商务、奢华、度假、服务式公寓等子品牌。
   - 各子品牌有不同的资本成本、目标客群与事件敏感度。

3. **供应链与劳动力**
   - 增加人员配置、采购合同、维护标准与劳资动荡风险。
   - 精益战略应带来短期节省，但过度使用会增加服务质量事故。

4. **资本市场（深化）**
   - 债务、信用评级、利息、股权发行、信贷紧缩、资产出售与收购要约。
   - 金融危机应迫使流动性管理，而不仅是压低需求。
   - 下一步：将全球金融危机与基准利率、借贷上限挂钩。

5. **竞争对手外交与并购**
   - AI 集团可结成区域联盟、发动价格战、收购物业或退出城市。
   - 玩家可在衰退期以估值区间收购困境酒店；详情页支持挂牌出售。
   - 详见 [HOTEL_OPERATIONS_DESIGN.md](./HOTEL_OPERATIONS_DESIGN.md) 第 3 节。

6. **广告与营销**
   - 集团品牌广告与单店广告并存；地铁、机场、OTA、财经媒体等形式对应不同客群。
   - 详见 [HOTEL_OPERATIONS_DESIGN.md](./HOTEL_OPERATIONS_DESIGN.md) 第 6 节。

7. **剧本目标**
   - 1990 香港开局战役。
   - 亚洲金融危机生存战役。
   - 中国入世增长战役。
   - 全球金融危机防守战役。
   - 疫情复苏战役。

## 交互与体验路线图

- 用任务式引导替代被动教程文案：查看城市、落子开店、调价、选战略、看新闻。
- **固定时间节奏**：每 5 秒真实时间推进 1 个游戏日；取消倍速按钮。玩家暂停思考，使用 **「快进下月」** 跳过整月。
- **资金 HUD 格式**：以 `$111,000 (+$11,000)` 显示，括号内为最近模拟日的净现金变化（收入减运营成本与利息）。
- 为城市档案增加历史时间线、需求驱动因素、本地风险与推荐酒店类型。
- 为引导流程增加任务完成奖励与步骤小结。
- 为档案增加城市专属插画或天际线缩略图。
- 增加解锁、危机升级、复苏、对手扩张等战役通知动画。
- 增加决策日志，让玩家理解现金、声誉、需求或城市指标为何变化。
- 增加战略地图模式：航线、区域集团、危机叠加、自有网络影响力。

## 视觉资产路线图

- 战略、事件类别、酒店档次、城市状态的扁平图标。
- 重大时代场景图：1990 年代香港、入世中国、2008 后金融、疫情封控、旅行复苏。
- 3/4/5 星、度假、机场酒店、综合体等简易 3D/等距酒店模型。
- 重大历史冲击与机遇的事件明信片。
- 航空走廊与区域需求流动的动画航线。

## 内容路线图

- 在现有亚太/美国焦点外扩展城市：伦敦、巴黎、迪拜、曼谷、悉尼、孟买、雅加达、台北已上线；未来目标包括澳门、马尼拉、柏林、多伦多、伊斯坦布尔、圣保罗。
- 为各城市增加人口、旅游、GDP 代理、机场容量、酒店供给的历史曲线。
- 用事件链替代一次性修正：危机预警 → 冲击 → 政策应对 → 复苏。
- 增加架空历史决策，让玩家缓解或放大历史结果。

## 里程碑

### 里程碑 1：扎实的 MVP 战役

- 任务式教程。
- 战略政策 UI 与平衡调优。
- 带后果说明的更佳事件 feed。
- 第一批视觉资产。
- 覆盖开局、建造、切换战略、事件弹窗的 E2E。

### 里程碑 2：区域宏大战略

- 城市档案。
- 品牌组合。
- 资本与债务。
- 竞争对手扩张地图。
- 聚焦亚洲的剧本目标。

### 里程碑 3：全球历史战役

- 25+ 城市。
- 事件链。
- 并购与外交。
- 多种胜利条件。
- 存档槽与战役总结。

## 开放设计问题

- 玩家是否应永远直接管控单店，还是逐步委托区域经理？
- 历史事件应按日期确定性触发、在窗口内半随机，还是剧本专属？
- 危机对休闲玩家与宏大战略玩家应有多严厉？
- Mapbox 是否继续作为主地图，还是 stylized 战略地图应成为默认以兼顾性能与美术方向？
